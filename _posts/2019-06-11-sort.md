排序分为冒泡排序

### 冒泡排序

将一个字符串里的字母逆序输出

```java
int array[] = new int[]{3, 2, 4, 8, 7, 9, 5, 1, 0};
int swap;
for (int i = 0; i < array.length - 1; i++) {
    for (int j = i + 1; j < array.length; j++) {
        if (array[i] > array[j]) {
            swap = array[j];
            array[j] = array[i];
            array[i] = swap;
        }
    }
}
```
